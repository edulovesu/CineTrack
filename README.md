# CineTrack — Personal Movie Watchlist App

A fully functional movie watchlist app built with vanilla JavaScript.

## Live Demo
[View Live Site](your-netlify-link-here)

## Features
- Add movies with title, genre and rating
- Mark movies as To Watch, Watching or Watched
- Delete movies from your library
- Search movies in real time
- Filter by genre and status
- Sort by newest, rating or A-Z
- Data persists across sessions using localStorage

## Tech Stack
HTML · CSS · Vanilla JavaScript · localStorage API
